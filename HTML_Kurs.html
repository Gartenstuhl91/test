Forms and Button













Eine Form kann man sich wie einen Rahmen vorstellen und alles was in diesem Rahmen enthakten ist, ob es Texte, Buttons usw sind, wird zusammen an die in dem Form eingebetteten Link gesendet.

Mit dem ein Input hat auch einen Type. In den Type lässt sich die Art des Inputs definieren. Von Text, password, color usw.

Mit dem Label, lassen sich Inputs beschriften. Sie müssen mit deren ID verknüpft werden. Steht das Label über dem Input, steht die beschreibung vor dem Text
